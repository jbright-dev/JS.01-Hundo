The Code

                    
                        //Get values from the page
//Controller Function
function getValues(){
    //get values from the page
    let startValue = document.getElementById("startValue").value;
    let endValue = document.getElementById("endValue").value;

    //validate data as int
    startValue = parseInt(startValue);
    endValue = parseInt(endValue);

    if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                
        //call generate numbers
        let numbers = generateNumbers(startValue, endValue);            
        //call display numbers
        displayNumbers(numbers);

    }else{
        alert("Please enter an integer");
    }

}

//Generate numbers from startValue to endValue
//Business Logic Function
function generateNumbers(sValue, eValue){
    
    //assign the variable numbers a value of an empty array
    let numbers = [];

    //we want to get all number from start to end
    for (let index = sValue; index <= eValue; index++) {
        
        //this will execute in a loop until index = eValue
        numbers.push(index);
        
    }

    return numbers;
}

//Display numbers with the even numbers being bold
//View Function
function displayNumbers(numbers){
    
    let templateRows = "";

    for (let index = 0; index < numbers.length; index++) {
        
        let className ="even";

        let number = numbers[index];

        if (number % 2 == 0) {
            className = "even";
        }else{
            className = "odd";
        }

        templateRows += `<tr><td class="${className}" >${number}</td></tr>
    }

    document.getElementById("results").innerHTML = templateRows;

}
                        }

The Code is structured into three separate functions:

1. The Controller Function
2. The Business Logic Function
3. The View Function

The Controller Function

The Controller Function serves as an intermediary that receives user input, delegates tasks to the business logic layer, and prepares the proper view to be sent back to the user.

For this application, the controller collects the start and end values from the page via the Start Value and End Value input boxes.

The values are then validated as proper integers by using the built-in parsing function in JavaScript. If the numbers are integers, the view is called, and the data is presented in the UI.



The Business Logic Function

The Business Logic Function serves as a set of rules and operations that define what the application should be doing.

For this application, the application declares a variable named "numbers" and assigns it the value of an empty array.

A for loop that iniitialzes a variable named index which is assigned the input start value. As long as the index is less than the end value, the index will increment upwards by one.



The View Function

The View Function, in this instance, represents a specific presentation of data or information to the user.

For this application, a variable named templateRows is declared and is assigned the value of an empty string.

A for loop iniitialzes a variable named index, which is assigned the value of zero. As long as the value of the index is less than the length of the numbers array, index will increment upwards by one

Within the loop, a variable named className is declared and is assigned the value of "even". A variable named number is declared and assigned the value of the index of the numbers array.

An If Statement indicates if the variable "number" has a modulus that is divisible by two, assign it the className even, else it is assigned the className odd.

The variable templateRows then utilizes a template literal to populate the HTML with the proper class name based on the number value.

Finally, the document is targeted, and the results ID is located within the inner HTML and is assigned the value of the current template row.
